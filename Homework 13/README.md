# Reflection
For this assignment I created a 3D scene using a model and five textured shapes. I used arrays and a loop to place the shapes in different positions and make them rotate. I also added interaction so when
the mouse is clicked, some of the shapes move to random positions. I struggled the most with getting the model to load and position correctly in 3D space. I also had some issues with textures and text not
showing at first. I used examples from the lessons and ChatGPT to help debug and figure out what was going wrong. Overall, I think my project turned out well and shows motion, texture, and interaction clearly.
